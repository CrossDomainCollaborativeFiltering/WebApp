Overview

Recommender Systems can be broadly classified into 3 categories. They are Content-Based Filtering, Collaborative Filtering and Hybrid Filtering. Content-Based Filtering is a technique of filtering where we use the raw data of the items considered for recommendation and recommend that to an end-user. Collaborative Filtering is a technique of filtering where we use the data from ther users who have used the item and rated the item and we recommend based on that. Hybrid as the name suggests is an amalgamation of both content based and Collabortive filtering.

Methodology

Three of the biggest problems in collaborative filtering are: scalability, Sparsity and Cold Start. To address all these issues we devised a methodology involving primarily 2 steps: Preprocessing which would be a one-time computation where the data can be stored in the secondary storage for future use and Querying which would compute the recommeded items for each user on the user database.
